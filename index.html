Discord Counter

Blocks page style modifications in moderator posts on 4chan.

Info

Moderators on 4chan sometimes like to mess with threads by including stylesheet data in their posts, which can cause the thread to not display, have a ridiculous background, etc. This script simply blocks that type of data from being loaded from inside posts.
